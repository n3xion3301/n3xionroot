# Supported Devices

## Currently Supported

### Samsung Galaxy Series
- **Galaxy S10 (SM-G973F)** - Android 9-12
  - Status: ✅ Tested
  - Method: Magisk + TWRP
  
- **Galaxy A52 (SM-A525F)** - Android 11-13
  - Status: ✅ Tested
  - Method: Magisk patched boot

### Google Pixel Series
- **Pixel 4a (sunfish)** - Android 11-13
  - Status: ✅ Tested
  - Method: Magisk + fastboot

### OnePlus Series
- **OnePlus 7 Pro (guacamole)** - Android 10-12
  - Status: ⚠️ Beta
  - Method: Custom recovery + Magisk

## Device Status Legend
- ✅ Tested - Fully working
- ⚠️ Beta - Works but needs more testing
- 🚧 WIP - Work in progress
- ❌ Broken - Known issues
